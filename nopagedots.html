Making My First Tweak: NoPageDots

This tutorial is made for iOS 10 so things may not be exactly the same in the future.

Things needed for this tutorial
  • theos installation
  • new project ready for your tweak
  • test device to test your tweak

Links

The first part of making a tweak is finding the classes to hook. A lot of tutorials skip this step but an easy way of doing this is using http://developer.limneos.net. This has a search bar which makes easy to find which header you are looking for.

The aim of this tweak will be to hide the page dots on the homescreen. After thinking about what you want to do use http://developer.limneos.net to search for the correct header. I'll give you a hint, the name of this header isn't simply SBPageDots. %hook SBIconListPageControl - (id)initWithFrame:(CGRect)frame { return SBisEnabled ? nil : %orig; } %end
